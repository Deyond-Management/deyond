# Privacy Policy

**Last Updated: [DATE]**

Deyond ("we," "our," or "us") is committed to protecting your privacy. This Privacy Policy explains how we collect, use, and protect your information when you use our mobile application ("Deyond Wallet" or "App").

## 1. Information We Collect

### 1.1 Information You Provide
- **Wallet Data**: Your wallet addresses (public keys only)
- **Preferences**: Language, currency, theme settings
- **Support**: Communications when you contact us

### 1.2 Information We Do NOT Collect
- Private keys or seed phrases
- Passwords or PINs
- Transaction history (stored locally only)
- Personal identification information

### 1.3 Automatically Collected Information
- Device type and operating system
- App version
- Crash reports and performance data
- Anonymous usage analytics

## 2. How We Use Your Information

We use information to:
- Provide and improve our services
- Display your wallet balances
- Send transaction notifications
- Fix bugs and improve performance
- Comply with legal obligations

## 3. Data Storage and Security

### 3.1 Local Storage
- All sensitive data is stored locally on your device
- Private keys are encrypted using AES-256-GCM
- We use secure storage (iOS Keychain / Android Keystore)

### 3.2 No Cloud Storage
- We do not store your private keys on any server
- Your recovery phrase is never transmitted

### 3.3 Security Measures
- End-to-end encryption for all communications
- Regular security audits
- Bug bounty program

## 4. Third-Party Services

We use the following third-party services:

| Service | Purpose | Data Shared |
|---------|---------|-------------|
| Alchemy/Infura | Blockchain RPC | Wallet addresses, transactions |
| CoinGecko | Price data | None |
| Sentry | Crash reporting | Device info, error logs |
| Analytics | Usage statistics | Anonymous events |

- Access your data

- Opt-out of analytics
